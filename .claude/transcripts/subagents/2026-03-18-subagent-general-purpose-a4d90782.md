## 1. Right-Click Options for FILES in Obsidian's File Explorer

When right-clicking a **file** (note) in the file explorer:

- **Open in new tab**
- **Open to the right** (split pane)
- **Open in new window** (pop-out window)
- ---
- **Set as bookmark** (previously called "Star" before the Bookmarks plugin replaced Stars ~v1.2)
- ---
- **Make a copy** (duplicates the file)
- **Rename** (inline rename; Obsidian auto-updates all internal links)
- **Delete** (behavior configurable: system trash, Obsidian .trash, or permanent)
- ---
- **Move file to...** (opens a search modal to pick a destination folder)
- ---
- **Open in default app** (requires "Open in default app" core plugin enabled)
- **Show in system explorer** (Finder on macOS, File Explorer on Windows; also requires "Open in default app" core plugin)
- **Reveal file in navigation** (scrolls/highlights the file in the sidebar)
- ---
- **Copy Obsidian URL** (copies an `obsidian://` protocol URI)

## 2. Right-Click Options for FOLDERS in Obsidian's File Explorer

When right-clicking a **folder**:

- **New note** (creates a note inside this folder)
- **New folder** (creates a subfolder)
- ---
- **Set as bookmark**
- ---
- **Duplicate** (duplicates the entire folder and contents)
- **Rename**
- **Delete**
- ---
- **Move folder to...** (search modal for new parent)
- ---
- **Show in system explorer** (requires "Open in default app" core plugin)
- **Reveal file in navigation**
- ---
- **Copy Obsidian URL**

Key differences: Folders have "New note" and "New folder" at the top. They do NOT have "Open in new tab/right/window" (folders cannot be opened as tabs). "Make a copy" is labeled "Duplicate" for folders.

## 3. Notable File Management Features

**File Explorer Toolbar:**
- New note button (pen icon) -- creates note in vault root
- New folder button (folder+ icon)
- Sort order dropdown: File name (A-Z / Z-A), Modified time (new-old / old-new), Created time (new-old / old-new)
- Collapse all button
- Auto-reveal active file toggle (auto-scrolls to highlight the currently open note)

**Drag and Drop:**
- Drag files between folders to move them
- Drag a file into an open note to create a `[[wikilink]]`
- Drag external files into the vault to import them
- Alt/Opt + click for multi-select, then drag to move together

**Bookmarks (formerly Stars):**
- Bookmark notes, folders, searches, headings, and blocks
- Dedicated "Bookmarks" sidebar panel
- Bookmarks can be organized into bookmark groups (sub-folders within the bookmarks panel)

**Delete Behavior (Settings > Files & Links):**
- System trash (default, recoverable)
- Obsidian trash (`.trash/` folder in vault)
- Permanently delete (no recovery)

**Other Settings:**
- Excluded files patterns
- Configurable attachment folder location
- "Detect all file extensions" toggle for non-markdown files
- Default location for new notes: vault root, same folder as current file, or a specified folder

**What Obsidian does NOT have natively (common plugin territory):**
- Pinning files to the top of the explorer
- Custom drag-and-drop sort order (requires "Custom Sort" plugin)
- File/folder coloring or custom icons (requires "Color Folders and Files" or "Iconize" plugins)
- File previews/thumbnails in the explorer (requires "Notebook Navigator" plugin)
- Filtering/hiding specific files beyond excluded patterns (requires "File Explorer++" plugin)
