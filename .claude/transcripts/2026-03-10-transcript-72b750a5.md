## Step 1: `GraphView.tsx` — Aggressive fcose params

In `runLayout`, replace the fcose block:

```ts
cy.layout({
  name: "fcose",
  animate: false,
  quality: "proof",
  idealEdgeLength: 280,    // was 180 — longer rest length spreads connected pairs
  nodeRepulsion: 75000,    // was 18000 — 4× stronger push
  edgeElasticity: 0.30,   // new — softer springs (default ~0.45); hub nodes stop acting as gravity wells
  gravity: 0.04,           // was 0.15 — barely enough to keep disconnected nodes on screen
  gravityRange: 5.0,       // new — extends the radius before gravity kicks in (default 3.8)
  numIter: 2500,           // new — ensures convergence at aggressive repulsion values
  fit: true,
  padding: 60,
  nodeDimensionsIncludeLabels: false,
} as cytoscape.LayoutOptions).run();
```

**Why each value:**
- `nodeRepulsion: 75000` — with 34 nodes and 116 edges, at 18000 the edge springs win. At 75000 nodes reach the boundary shell before edges pull them back.
- `idealEdgeLength: 280` — must increase in proportion to repulsion, otherwise edges yank nodes together at the new distances.
- `edgeElasticity: 0.30` — mirrors Obsidian's weak link force (~1/degree per edge). Hub nodes with many edges are no longer gravity wells.
- `gravity: 0.04` — just enough to stop fully disconnected nodes from flying off-screen.
- `gravityRange: 5.0` — person nodes with 1-2 `authored-by` edges can sit at the periphery without being yanked back.
- `numIter: 2500` — aggressive repulsion needs more iterations to fully settle (~200ms layout, invisible since `animate: false`).

## Step 2: `uiStore.ts` — `hiddenEdgeTypes` state

Follow the exact same `Set<string>` pattern as `treeExpandedFolders`.

**Add to `UIState` interface:**
```ts
hiddenEdgeTypes: Set<string>;
toggleEdgeType: (rel: string) => void;
```

**Add to store initializer:**
```ts
hiddenEdgeTypes: new Set<string>(),

toggleEdgeType: (rel: string) =>
  set((s) => {
    const next = new Set(s.hiddenEdgeTypes);
    if (next.has(rel)) next.delete(rel);
    else next.add(rel);
    return { hiddenEdgeTypes: next };
  }),
```

Default is empty Set — no edges hidden. User opts in to filtering.


**Click-outside handler:**
```ts
useEffect(() => {
  if (!edgeFilterOpen) return;
  const handler = (e: MouseEvent) => {
    if (popoverRef.current && !popoverRef.current.contains(e.target as Node)) {
      setEdgeFilterOpen(false);
    }
  };
  document.addEventListener("mousedown", handler);
  return () => document.removeEventListener("mousedown", handler);
}, [edgeFilterOpen]);
```

**New JSX — insert after the layout `<select>` and before the `<span style={{ flex: 1 }} />`:**
```tsx
<div ref={popoverRef} style={{ position: "relative" }}>
  <button
    className={hiddenEdgeTypes.size > 0 ? "active" : ""}
    onClick={() => setEdgeFilterOpen((o) => !o)}
    title="Filter edge types"
  >
    {hiddenEdgeTypes.size > 0
      ? `Edges (${edgeTypes.length - hiddenEdgeTypes.size}/${edgeTypes.length})`
      : "Edges"}
  </button>
  {edgeFilterOpen && (
    <div className="edge-filter-popover">
      {edgeTypes.map((rel) => (
        <label key={rel} className="edge-filter-item">
          <input
            type="checkbox"
            checked={!hiddenEdgeTypes.has(rel)}
            onChange={() => toggleEdgeType(rel)}
          />
          {rel}
        </label>
      ))}
    </div>
  )}
</div>
```
