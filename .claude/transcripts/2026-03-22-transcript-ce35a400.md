## Assistant



Yes, Canvas already supports post-it style observations. The **text node** type is exactly that — a free-form card you can place anywhere on the canvas. You can:

- **Add text cards** via the bottom toolbar ("Add Card" button)
- **Edit content** by double-clicking
- **Color-code** them with border and background colors (6 preset colors)
- **Resize** them freely
- **Connect** them to other nodes (file references, links, groups, other text cards) with labeled edges

So you can drop text cards as observations/annotations next to file nodes that reference your actual notes, connect them with arrows, and color-code them by category — effectively using them as post-its on a visual board.
