### High-Impact Visual Upgrades

**1. Metadata as styled pills/chips instead of a table**
Replace the plain table with inline chips: a colored type badge (e.g., "experiment" in a teal pill), tags as individual rounded chips, status as a colored dot + label. This is what Notion/Linear do — compact, scannable, visually rich. The collapsible "Metadata" header goes away; metadata lives right below the title as a natural part of the note header.

**2. Typography upgrade for preview mode**
Switch from monospace to a proportional serif or sans-serif font for the rendered markdown body (e.g., Inter, Source Serif, or system serif). Bigger headings with more weight contrast, tighter paragraph spacing, and proper `max-width` (~720px) with centered content for comfortable reading. Think Medium/iA Writer reading experience.

**3. Heading accent bar instead of red color**
The coral/red `##` headings are functional but flat. Consider a subtle left border accent bar on headings (like Notion's colored heading indicators) or gradient underlines. Less "raw markdown" feel, more polished document.

**4. Frontmatter-to-hero section**
Turn the note title + type + tags area into a "hero" header: large title text, a subtle type icon or emoji beside it, tags underneath, source as a small attribution line. Gives each note a distinct identity at a glance.

**5. Better Related Notes section**
Replace the plain list with cards or a mini-graph. Each related note could show as a small card with its type color, title, and relationship badge — or even a tiny force-directed subgraph of direct neighbors (you already have Cytoscape).

---

### Medium-Impact Polish

**6. Smooth transitions between edit/preview**
Fade or crossfade when switching modes instead of hard swap. Makes the toggle feel intentional rather than jarring.

**7. Reading progress / scroll indicator**
A thin accent-colored progress bar at the top of the editor panel showing scroll position. Subtle but adds dynamism.

**8. Syntax highlighting in code blocks (preview)**

**9. Floating TOC sidebar**
For longer notes, a mini table-of-contents floating on the right edge (auto-generated from headings). Clickable to jump to sections.

**10. Empty state illustration**
When no note is selected, show a light illustration or the BrainMap logo instead of blank space.
